# AIminer - AI Minecraft Bot Project

AIマインクラフターボット 総合作業場

## プロジェクト概要

マインクラフト（Paper 1.21）でAIボットが自動で採掘などのタスクを実行するシステムです。
データパックとプラグインを組み合わせて、AIサーバーと連携した自律型ボットを実現します。

## リポジトリ構造

```
AIminer/
├── plugin/              # プラグイン本体
│   └── AIminer/         # Paper 1.21プラグインのソースコード
├── datapack/            # データパック本体
│   └── manekinwalk_datapack/  # ボット制御用データパック

```

## 主な機能

### データパック側の機能
- **ボット召喚**: 見た目と移動用エンティティの召喚
- **木を掘る**: 指定座標の木を自動採掘
- **石を掘る**: 指定座標の石を自動採掘
- **移動**: 指定座標へ自動移動

### プラグイン側の機能（開発予定）
- **脳ファイル管理**: ボットの視覚・聴覚情報を記録
- **AIサーバー連携**: 脳ファイルをAIサーバーに送信して行動決定
- **タスク実行**: AIの指示に基づいてデータパック関数を実行
